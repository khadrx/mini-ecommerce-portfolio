# Mini E-commerce Portfolio

متجر إلكتروني مصغر – مشروع بورتفوليو Full-Stack

**Live Demo:** https://mini-ecommerce-portfolio.vercel.app  
**GitHub:** https://github.com/khadrx/mini-ecommerce-portfolio

### المميزات
- صفحة رئيسية مع عرض المنتجات
- صفحة تفاصيل المنتج
- سلة تسوق ديناميكية (إضافة، حذف، تعديل الكمية)
- أيقونة سلة في الهيدر مع العدد
- صفحة checkout مع نموذج بيانات + ملخص الطلب
- toast notifications (Sonner) لكل إجراء
- Footer احترافي مع اسمك ورابط GitHub
- RTL عربي كامل + Dark/Light mode جاهز

### التقنيات المستخدمة
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS + shadcn/ui
- Zustand (state management للسلة)
- Sonner (notifications)
- Next/Image (تحسين الصور)
